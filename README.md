ğŸ® How to Play
------------------------------------------------------------
1. Select "2 Players" or "vs Computer" mode.
2. Click a square to place your mark (X or O).
3. First player to align 3 marks wins.
4. Use Undo to go back one move.
5. Click Refresh to start a new game.
